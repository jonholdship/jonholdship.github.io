Character Sheet



A D&D 3.5e character sheet for Android

Character Sheet is an open source android app written in Java. It provides a character sheet for 3.5e D&D players that fully utilises the possibilities of keeping your character sheet on your phone. Most available apps reproduce the classic paper character sheet, putting all information on screen regardless of importance; Character Sheet displays only what is necessary. For example, skills are given as a list of skill names and total modifiers on a simple scrolling page, meaning that you only see what you need 99% of the time. When more detail is required, clicking a skill brings up a detailed page with all modifiers and the SRD rules for that skill.

The other clear advantage to an app version of your character sheet is automation. Change your Constitution, watch your HP change. Change class and watch your base attack bonuses and class skills change without any further input.

Character Sheet source code is available here. A spell tracker is yet to be implemented and a Google Play store link will be available as soon as that final feature is implemented.
